\subsection{Sessionkonzept \& History}
Das Sessionkonzept ermöglicht die Suche als Prozess. Jede Session verfügt über eine eigene History. Innerhalb einer Session wird jede Suchanfrage und jeder angezeigte Treffer automatisch von der Software als History gespeichert.
